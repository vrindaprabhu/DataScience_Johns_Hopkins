Including the required library and reading the datasets required

The raw_df is a data frame constructed by reading directly from csv provided.It is then converted into data table format raw_table. The active_df is data frame having removed all NA values.It is then converted into data table format active_table. Also date in active_df is converted from factor to Date format.

library(data.table)
library(ggplot2)

raw_df <- read.csv('activity.csv')
raw_table <- data.table(raw_df)

active_df <- raw_df[complete.cases(raw_df),]
active_df[,'date'] <- as.Date(active_df[,'date'], "%Y-%m-%d")
active_table <- data.table(active_df)
